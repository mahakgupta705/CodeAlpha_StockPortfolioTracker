# 📈 Stock Portfolio Tracker

A simple Python-based stock portfolio tracker built as part of the **CodeAlpha Python Programming Internship**.

## 📌 Description

This program allows users to enter stock names and quantities, calculates the total investment value based on predefined stock prices, and displays a portfolio summary. Users also have the option to save the summary to a text file.

## ✨ Features

- Predefined dictionary of stock prices
- User inputs stock name and quantity
- Automatically calculates total value per stock
- Combines quantities if the same stock is entered multiple times
- Displays a full portfolio summary
- Calculates total investment amount


## 🛠 Concepts Used

- Dictionaries
- Input/Output handling
- Basic arithmetic operations


## 🚀 How to Run

1. Make sure Python is installed on your system.
2. Clone this repository:
   ```bash
   git clone https://github.com/your-username/CodeAlpha_StockPortfolioTracker.git
   ```
3. Navigate to the project folder and run:
   ```bash
   python stock_tracker.py
   ```
4. Enter stock names and quantities as prompted. Type `done` when finished.
